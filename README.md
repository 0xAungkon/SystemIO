# **SystemIO**

### Features:

1.  Have System Monitor  
    1\. Cpu  
    2\. Ram  
    3\. Storage  
    4\. User's  
    5\. System Info
2.  Make Web Cli
3.  Have Take Manager
4.  Have Nginx Manager
5.  Have Docker Manager
6.  Proxy Manager
7.  Dns Server

---

### Tech:

1.  Sqlite3 (Database)
2.  Fastapi Python (Backend)
3.  React (Frontend)
